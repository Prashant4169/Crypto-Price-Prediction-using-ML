# Crypto-Price-Prediction-using-ML

A machine learning project that predicts Bitcoin (BTC/USD) prices using historical cryptocurrency market data.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Machine Learning model training
- Price prediction interface using Streamlit
- Interactive web application

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
